# RTOS Task Scheduler Report

## Abstract
This project demonstrates multitasking using FreeRTOS on an ESP32.

## Objectives
- Understand RTOS concepts
- Implement concurrent tasks
- Learn task scheduling

## Hardware
- ESP32

## Software
- Arduino IDE
- FreeRTOS

## Methodology
1. Create multiple tasks.
2. Assign priorities.
3. Execute tasks concurrently.
4. Monitor execution through Serial Monitor.

## Conclusion
FreeRTOS enables efficient multitasking in embedded systems.
